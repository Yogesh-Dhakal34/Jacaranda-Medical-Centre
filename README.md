# Jacaranda Blue-Mist Medical Centre

A responsive, multi-page healthcare website built with semantic HTML, modern CSS, and vanilla JavaScript.

## 🔗 Live Demo
[View Live Site] (https://yogesh-dhakal34.github.io/Jacaranda-Medical-Centre/)

## 📋 Features
* Responsive layout for mobile, tablet, and desktop screens.
* Interactive appointment booking form.
* Searchable directory for medical specialists.
* Accessible design with semantic HTML tags.

## 🛠️ Technology Stack
* HTML5
* CSS3
* Vanilla JavaScript

## 📂 Project Structure
* `index.html` - Homepage with hero section and site navigation.
* `services.html` - Catalog of medical specialties.
* `doctors.html` - Interactive doctor directory.
* `resources.html` - Patient portal and appointment form.
* `about.html` - History and mission of the center.

## 🚀 How to Run Locally
1. Clone the repository to your computer.
2. Open the `index.html` file in your web browser.
